# To-Do List Manager

This is a website to help organise tasks and sort them at different levels of urgency. Is was made with **Python**, **Flask**, **Bootstrap 5** and more.

## Features
* **Create Tasks**: give them a title a description and an urgency level
* **Set Urgency Levels**: Set tasks to be of high, medium or low priority
* **Edit Tasks**: You can cahnge a to-do's content even after creating it
* **Uses Simple Design**: Intuitive design with minimal clutter

## Project Structure
* **Backend**: Python, Flask
* **Frontend**: Jinja, Bootstrap 5, CSS
* **Data Storage**: Database with SQLAlchemy
